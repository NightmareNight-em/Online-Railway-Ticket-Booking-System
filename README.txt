# Online Railway Ticket Booking System


 Run reset.sh to create essential data files:
    >sh reset.sh 

Compile both the files

Ports of server and client can be changed (Make sure both are same)
ip address in client under variable name server_ip can also be changed.

run the server:
    >./server

run the client:
    >./client

Whenever asked for enter any key to return press only numeric key

If stuck run reset script and try again

On putting wrong ID can lead to infinte loop
